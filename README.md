Linkage Disequilibrium and Haplotype Analysis (Study 139_17)📌 Project OverviewThis repository contains the statistical analysis pipeline for genotypic data, focusing on identifying haplotype frequencies and mapping Linkage Disequilibrium (LD). The project utilizes Expectation-Maximization (EM) models to estimate haplotype probabilities in populations where the gametic phase is unknown.🛠️ Technologies and LibrariesThe analysis was developed in R, utilizing specialized statistical genetics packages:haplo.stats: For modeling haplotype frequencies via the EM algorithm.genetics: Processing genotype objects and calculating LD statistics ($D'$ and $r^2$).LDheatmap: Graphical visualization of recombination block structures.xlsx: Integration and reading of biological databases.🧬 Analysis WorkflowPre-processing: Converting alleles to numerical format (0, 1, 2), prioritizing the minor allele.Descriptive Statistics: Summarizing genotypes and identifying missing data (miss.val).EM Estimation: Running haplo.em to infer the haplotype composition of the sample.LD Calculation: Generating correlation matrices between loci.Visualization: Plotting heatmaps to identify genomic linkage blocks.📊 Key DeliverablesDetailed matrices for $D'$ and $r^2$ statistics.Heatmap plots with grayscale gradients to identify genomic proximity.Haplotype frequency estimates with robust handling of missing data.💡 Strategic InsightGiven the statistical complexity of genomic data, this pipeline emphasizes data integrity and reproducibility. By mapping LD blocks, we can significantly reduce dimensionality in Genome-Wide Association Studies (GWAS) and identify the most informative SNP markers for clinical or research outcomes.
